# oss-minesweeper
25-2 OSS(103) Pygame 기반으로 구현한 Minesweeper 과제 프로젝트입니다.

기본적인 지뢰찾기 규칙을 따르며, 플레이 편의성을 높이기 위한 여러 추가 기능을 포함하고 있습니다.

## 0. 기본적인 플레이
기본적인 플레이는 마우스로 이루어집니다.
- 좌클릭: 셀 열기
- 우클릭: 깃발 표시
- 중클릭: 인접 셀 표시

열린 셀의 숫자는 인접 셀에 지뢰가 몇 개나 있는지를 나타냅니다.

모든 안전한 셀을 열면 게임 클리어, 지뢰가 있는 셀을 열면 게임 오버입니다.



## 1. 난이도 선택
헤더의 버튼을 이용해 난이도를 선택할 수 있습니다.
| 난이도 | 보드 크기 | 지뢰 개수 |
|------|----------|----------|
| Beginner | 10 × 8 | 10 |
| Intermediate | 18 × 14 | 40 |
| Advanced | 24 × 20 | 99 |


## 2. 힌트
`H` 키 또는 상단의 `힌트` 버튼을 이용해 힌트를 사용할 수 있습니다.

힌트는 안전한 셀 1개를 초록색 테두리로 표시합니다.

게임 중 최대 2회 사용 가능합니다.



## 3. 숫자 셀 자동 열기(Chording)
숫자 셀에서 `Shift + 우클릭`으로 자동 열기가 가능합니다.

주변의 깃발 개수가 숫자와 일치하면 깃발이 설치되지 않은 셀이 자동으로 열립니다.

실제 지뢰찾기 게임의 chording 동작을 구현한 기능으로, 깃발이 설치되지 않은 셀에 지뢰가 있다면 게임 오버입니다.



## 4. 최단 기록 저장
게임 클리어 시 소요시간을 기록합니다.

기존 기록보다 빠를 경우 자동으로 갱신됩니다.

최고 기록은 `highsocre.txt` 파일에 저장되며, 헤더에 표시됩니다.



## 5. 게임 종료 및 재시작
게임 클리어 또는 게임 오버 시 표시되는 오버레이에 재시작 버튼을 추가하였습니다.

재시작 버튼 또는 `R`키로 재시작 할 수 있습니다.
